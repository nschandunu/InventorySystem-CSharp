SK Inventory Management System  
Final Project – Group AC  
Developed using C# WinForms and SQL Server

-----------------------------------------------------------
How to Run the Application:

1. Make sure you have SQL Server installed and running.
2. Place the provided database files (dbIMS.mdf and dbIMS_log.ldf)
   in the following location:
   → D:\

3. Open the solution file (SmartInventoryTracker.sln) in Visual Studio.
4. Check the database connection string in the code:
   - It is configured to look for the database in D:\

   Example connection string:
   @"Data Source=(LocalDB)\MSSQLLocalDB;AttachDbFilename=D:\dbIMS.mdf;Integrated Security=True"

5. Build and run the project.

6. Use the login screen to access the system:
   → **Username:** admin  
   → **Password:** admin
